# YtDlpGui - simple .net GUI for yt-dlp

![](main_window.png)

### Features

- **Modified aria2**, allowing you to download a file at the maximum bandwidth speed of your internet connection
- Possible support for services other than YouTube (tested only on vk)
- Ability to download video and audio both separately and combined
- Proxy support (format `http://login:pass@ip:port`)
